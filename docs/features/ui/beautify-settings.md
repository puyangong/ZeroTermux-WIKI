# 美化设置

美化设置用于调整终端界面外观，包括背景、菜单样式和相关显示效果。

## 入口

```text
美化/UI 功能 > 美化设置
```

## 常见用途

- 调整终端显示风格。
- 配合字体设置改善阅读效果。
- 设置左/右菜单背景图片。
- 开启或关闭部分视觉效果。

## 背景图片

`zt` 命令也提供设置终端背景图的能力：

```bash
zt backgroundimage /data/data/com.termux/files/home/.img/back.jpg
zt bgi /data/data/com.termux/files/home/.img/back.jpg
```

图片路径必须是完整路径。如果路径无效，ZeroTermux 会提示背景图片路径错误。
