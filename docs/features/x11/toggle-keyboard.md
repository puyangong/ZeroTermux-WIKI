# 显示/隐藏 X11 键盘

X11 键盘入口用于控制图形界面中的软键盘显示。

## 菜单入口

```text
X11 功能 > 显示键盘
X11 功能 > 隐藏键盘
```

## 命令方式

```bash
zt x11keyboardshow
zt x11kbs
zt x11keyboardhide
zt x11kbh
```

## 常见问题

### 命令执行后没反应

确认 X11 通道已启动，并且当前焦点在 X11 窗口内。

### 物理键盘和软键盘冲突

可以进入 Termux 设置或 ZeroTermux 设置调整软键盘行为。如果外接键盘可用，建议隐藏软键盘以获得更大的显示区域。
