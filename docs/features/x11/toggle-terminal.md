# 显示/隐藏 X11 终端

ZeroTermux 提供菜单和 `zt` 命令两种方式控制 X11 终端显示状态。

## 菜单入口

```text
X11 功能 > 显示终端
X11 功能 > 隐藏终端
```

## 命令方式

```bash
zt x11commandshow
zt x11xs
zt x11commandhide
zt x11xh
```

`x11xs` 是显示终端的短命令，`x11xh` 是隐藏终端的短命令。

## 注意事项

- 此功能需要 X11 已经启动。
- 如果命令没有效果，先执行 `zt x11status` 查看内部通道状态。
- 修改显示状态后，部分应用可能需要重新聚焦窗口。
