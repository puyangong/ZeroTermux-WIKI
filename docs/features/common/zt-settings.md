# ZT 设置

ZT 设置用于调整 ZeroTermux 的专属行为，不同于原版 Termux 设置。

## 入口

```text
常用功能 > ZT设置
```

## 常用设置

| 设置 | 说明 |
| --- | --- |
| 使用习惯 | 在 Termux 风格和 ZeroTermux 风格之间切换 |
| 双击功能 | 设置双击终端时执行键盘、禁用或自定义命令 |
| 左/右栏目背景图片 | 从 `~Ztinfo/menuBack/` 读取 `left.png` 和 `right.png` |
| XML 菜单配置 | 控制是否使用配置文件设置左侧菜单 |
| 底部键盘 | 自定义终端底部额外按键 |

## 使用习惯

Termux 使用习惯：

- 双击滑动不弹出工具箱。
- 音量键不用于弹出左右菜单。

ZeroTermux 使用习惯：

- 双击或边缘操作可弹出工具箱。
- 音量键可用于弹出左右菜单。

## 底部键盘

底部键盘入口会修改 Termux 的额外按键配置，保存后需要重启应用生效。

如果配置格式错误，会提示“格式错误”。建议先从默认配置开始改，每次只改一小段。
