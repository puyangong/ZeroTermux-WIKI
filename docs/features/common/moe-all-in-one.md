# MOE 全能

MOE 全能是 ZeroTermux 常用功能中的脚本入口，适合需要快速拉起一组集成工具的用户。

:::warning
TMOE 已被归档，若无使用及恢复 ZeroTermux 环境的经验，请谨慎使用。
:::

## 入口

```text
常用功能 > MOE全能
```

点击后会向当前终端发送对应命令或脚本。脚本具体内容会随 ZeroTermux 版本和线上资源变化。

## 使用前检查

1. 已完成 ZeroTermux 首次启动。
2. 已授予内部存储权限。
3. 软件源可用，可以正常执行 `pkg update`。
4. 网络能访问脚本所需地址。

## 运行失败怎么办

- 先执行 `pkg update`，确认包管理器可用。
- 切换到更稳定的软件源。
- 重新打开 ZeroTermux，再运行一次。
- 如果脚本提示缺少目录，检查 `/sdcard/xinhao/` 是否已创建。

:::warning
MOE 全能可能会改动当前环境。运行前建议先备份重要容器。
:::
