# Zero 功能

Zero 功能是 ZeroTermux 在原版 Termux 之外加入的一组扩展能力。不同版本中入口可能会拆分到“常用功能”“配置终端”“线上功能”“ZT功能”等分类。

## 功能概览

| 分类 | 示例 |
| --- | --- |
| 常用功能 | 切换源、容器切换、备份/恢复、发行版本、QEMU |
| X11 功能 | X11 设置、终端显示/隐藏、键盘显示/隐藏、VNC |
| 美化/UI | 悬浮窗口、美化设置、字体、全屏、动画、视频背景 |
| 线上功能 | 在线脚本、论坛、下载站、公共仓库 |
| 配置终端 | ADB Shell、底部键盘、启动脚本、命令定义 |
| ZT 功能 | 模块安装、远程连接、Web 数据、短信/通讯录、语言切换 |
