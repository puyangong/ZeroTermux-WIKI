# VNC

ZeroTermux 内置 VNC 连接入口，可快速连接本机或自定义 VNC 服务。

## 菜单入口

```text
X11 功能 > VNC
```

菜单包含：

| 选项 | 说明 |
| --- | --- |
| 快速VNC | 连接 `127.0.0.1:5901`，默认密码 `123456` |
| 自定VNC | 手动输入地址、端口和密码 |

## 命令方式

也可以使用 `zt vnc`：

```bash
zt vnc 127.0.0.1_5900_admin_admin
zt vnc 127.0.0.1_5900_admin_none
zt vnc 127.0.0.1_5900_none_admin
zt vnc 127.0.0.1_5900_none_none
```

格式：

```text
zt vnc 地址_端口_用户名_密码
```

用户名或密码为空时使用 `none`。地址和端口不能为空。

## 常见问题

- 连接失败：确认 VNC 服务端正在运行。
- 密码错误：检查服务端密码设置。
- 连接本机桌面：通常使用 `127.0.0.1` 和服务端实际端口。
