# 打开目录

“打开目录”用于快速打开 ZeroTermux 管理的常用文件夹，便于导入备份、字体、镜像和配置文件。

## 入口

```text
ZT 功能 > 打开目录
```

## 常用目录

| 目录 | 用途 |
| --- | --- |
| `/sdcard/xinhao/data/` | 备份/恢复包 |
| `/sdcard/xinhao/apk/` | 临时 APK |
| `/sdcard/xinhao/command/` | 自定义命令 |
| `/sdcard/xinhao/font/` | 字体 |
| `/sdcard/xinhao/iso/` | ISO 镜像 |
| `/sdcard/xinhao/qemu/` | QEMU |
| `/sdcard/xinhao/share/` | 共享文件 |

## 看不到目录

1. 确认已授予内部存储权限。
2. 完成首次启动中的创建目录步骤。
3. 如果目录被手动删除，重新进入引导或手动创建 `/sdcard/xinhao/` 下的子目录。
