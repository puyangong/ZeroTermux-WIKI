# 下载站

下载站提供已编译 APK、数据包或其他 ZeroTermux 相关资源。

## 入口

```text
线上功能 > 下载站
```

项目 README 中提供的 APK 下载地址：

```text
https://od.ixcmstudio.cn/repository/main/ZeroTermux/
```

## 下载建议

- 不要直接安装来源不明的 APK。
- 下载恢复包后放到 `/sdcard/xinhao/data/`。
- 恢复包必须确认适用于 ZeroTermux。
- 重要环境恢复前先备份当前容器。

:::warning
恢复包、数据包、zip 包内容不可由 ZeroTermux 完全控制。使用前请确认来源可信。
:::
