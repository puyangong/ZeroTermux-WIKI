# 公共仓库

公共仓库入口用于访问 ZeroTermux 相关开源仓库或资源仓库。

## 入口

```text
线上功能 > 公共仓库
```

## 反馈建议

提交问题时尽量提供：

- ZeroTermux 版本。
- Android 版本和设备型号。
- 复现步骤。
- 错误截图或终端输出。
